YouTube Events
Binding JS to the YouTube Player

YouTube Events allows for developers to bind callbacks to iFramed YouTube videos. In the examples we'll show you how to bind simple console logging, however the code does not limit you to just that. You can bind any JavaScript you'd like!

An example

Before we begin, let's see an example of what this can do! Play, pause, or do whatever you'd like to the YouTube video below. Underneath the YouTube video, there is a table that shows all of the actions that the user (that's you!) is doing to the video. Then, view the code that is outputting to your browser's console.

Video Title Action Video Position
Play the video above to see something here!

The code

Below is the code that output to your browser's console for the example above

YouTubeEvents.bind('load', function(data) {
    console.log(
        data.video.title,
        'LOAD'
    );
});

YouTubeEvents.bind('initial', function(data) {
    console.log(
        data.video.title,
        '1st PLAY'
    );
});

YouTubeEvents.bind('replay', function(data) {
    console.log(
        data.video.title,
        'REPLAY at',
        data.video.currentTime + ' seconds'
    );
});

YouTubeEvents.bind('play', function(data) {
    console.log(
        data.video.title,
        'PLAY at',
        data.video.currentTime + ' seconds'
    );
});

YouTubeEvents.bind('pause', function(data) {
    console.log(
        data.video.title,
        'PAUSE at',
        data.video.currentTime + ' seconds'
    );
});

YouTubeEvents.bind('complete', function(data) {
    console.log(
        data.video.title,
        'COMPLETE'
    );
});
    
// Milestone event, with percentage viewed as the value
YouTubeEvents.bind('milestone', function(data) {
    console.log(
        data.video.title,
        'MILESTONE ' + data.milestone + '% at',
        data.video.currentTime + ' seconds'
    );
});

// Cued up in the playlist
YouTubeEvents.bind('cued', function(data) {
    console.log(
        data.video.title,
        'cued'
    );
});

// Buffering event
YouTubeEvents.bind('buffer', function(data) {
    console.log(
        data.video.title,
        'BUFFERING at',
        data.video.currentTime + ' seconds'
    );
});

// Seeked event
YouTubeEvents.bind('seek', function(data) {
    console.log(
        data.video.title,
        'SEEKED to ',
        data.video.currentTime + ' seconds'
    );
});
